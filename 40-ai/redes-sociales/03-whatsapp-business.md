## 5. Flujo de calificación de leads (para el equipo comercial)

```
Mensaje entra
     │
     ├─ ¿Es cotización? → Pedir datos → Generar cotización (< 24h)
     │
     ├─ ¿Es consulta de folio? → /folio → Buscar en sistema
     │
     ├─ ¿Es urgencia? → /urgente → Llamar en 10 min
     │
     ├─ ¿Es queja o problema? → Escalar a servicios@inymet.com.mx
     │
     └─ ¿Es prospecto nuevo? → Calificar:
           1. ¿Qué industria?
           2. ¿Cuántos instrumentos?
           3. ¿Tiene auditoría próxima?
           4. ¿Tiene proveedor actual?
           → Si califica: agendar llamada técnica
```

## 6. Integración futura con n8n (según Estrategia Integral)

La estrategia menciona n8n para automatizar el flujo de ventas. Conexión sugerida:

```
WhatsApp Business API (Cloud API de Meta)
     ↓
n8n workflow
     ├─ Si mensaje contiene folio → Query a BD → Respuesta automática
     ├─ Si dice "cotización" → Crear ticket en HubSpot + notificar vendedor
     ├─ Si es fuera de horario → Mensaje de ausencia + añadir a CRM
     └─ Si nuevo contacto → Crear contacto en HubSpot automáticamente
```

Para activar esta integración:
1. Registrar en Meta Business Suite
2. Crear app con WhatsApp Cloud API
3. Conectar con n8n usando el webhook de WhatsApp
4. Crear workflows en n8n para cada caso de uso
